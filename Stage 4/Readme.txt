Terminal Ticketing Software Readme:

To compile and run the program, either:
  -Open the TerminalTicketingGUI folder as a java NetBeans project and run the Main.java class file, or
  -Run the distributed jar file found in the TerminalTicketingGUI/dist folder (through the java -jar command or simply by double clicking the executable).

To utilize the test data for the program:
  -Open the Load Data menu tab
  -Type the appropriate path name for the path to the testData.txt file in the TerminalTicketingGUI/src/MainPackage folder
  -Click "Load Data" and use the data within the program (see the user guide for more information)
